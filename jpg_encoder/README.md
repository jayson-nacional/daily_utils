# Dependencies  
1. imagemagick
2. libheif

# Linux / MacOS
1. Copy the bash script (**jpg_encoder.sh**) or directly clone the repository
2. Make the script executable. ```chmod +x jpg_encoer.sh```
3. (Optional) Copy to path to execute anywhere ```sudo mv jpg_encoder.sh /usr/local/bin/[your_script_alias]```

# Windows  
1. Copy the powershell script (**jpg_encoder.ps1**)
2. Run the script where the media files are located  

*Note: Encoded files are to be found on the compressed folder*
